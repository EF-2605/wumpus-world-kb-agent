# Wumpus World - Knowledge Based Agent

This project is an implementation of the classic Wumpus World problem from Artificial Intelligence using propositional logic and resolution refutation.

## Features

* Knowledge-Based AI Agent
* Propositional Logic Inference
* Resolution Refutation
* Interactive Cyber-Themed UI
* Automatic Safe Cell Detection
* Auto Play Mode
* Dynamic Grid Generation

## Technologies Used

* HTML5
* CSS3
* JavaScript

## How to Run

1. Download the project files
2. Open `wumpus_world.html` in any browser

OR

Use the live deployed version on Vercel.

## Game Elements

* 🤖 Agent
* 🕳 Pit
* 👹 Wumpus
* 💰 Gold
* 💨 Breeze
* ☠ Stench
* ✨ Glitter

## AI Concepts Used

* Knowledge Base (KB)
* Conjunctive Normal Form (CNF)
* Propositional Logic
* Resolution Refutation
* Logical Inference
